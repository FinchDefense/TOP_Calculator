# TOP_Calculator
A fully functional calculator web app with basic arithmetic operations built as part of The Odin Project curriculum.

Features:
- Addition, subtraction, multiplication, division, percentage
- Chain calculations (e.g., 5 + 3 - 2 = 6)
- Clear (C) and All Clear (AC)
- Division by zero handling
- Result displayed → fresh start on next number

What I learned/praticed:
- DOM manipulation (creating, appending, and removing elements)
- Event delegation (listening for clicks on parent elements)
- State management (tracking numbers, operator, and display state with variables)
- Conditional logic (if/else statements for different button behaviors)
- Function composition (breaking down operations into reusable functions)
- Data validation (checking for NaN, division by zero, decimal points)
